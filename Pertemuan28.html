Aritmetic Operators

Arimetic Operator adalah operasi aritmatika dasar

Operator Deskripsi
+ Tambah / Addition
- Kurang / Subtraction
* Kali / Multiplication
** Pangkat / Exponentiation
/ Bagi / Division
% Sisa Bagi / Modulus
++ Peningkatan / Increment
-- Penurunan / Decrement

Assignment Operators

Assigment Operator adalah operator yang memberikan nilai pada sebuah variable pada javascript.

Di bawah ini adalah table contoh dimana nilai sebuah variable adalah 1

Operator Penerapan Hasil
= variable = 1 1
+= variable += 2 3
*= variable *= 2 2
**= **= 2 1
/= Variable /= 2 0.5
%= variable %= 2 1
