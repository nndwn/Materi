HTML DOM (Document Objeck Model)

Sebelumnya kita sudah memahami membuat program sederhana dengan menggunakan javascript function yang dijalankan melalui console pada browser namun bagaimana membuat input dan output untuk menampilkan hasil nilai dari function dengan menggunakan HTML agar tampak lebih interaktif ? , HTML DOM adalah teknik manipulasi HTML dimana disaat Halaman Web Dimuat atau tampil , browser membuat Document Object Model. DOM itu adalah struktur HTML berupa document yang dapat dibaca oleh browser. berikut adalah contoh gambaran DOM

DOM HTML YANG DAPAT DILAKUKAN

  1. Dapat Mengganti atau mengubah Element HTML pada Halaman Web.
  2. Dapat Mengganti atau mengubah Attribut HTML pada Halaman Web.
  3. Dapat Mengganti atau mengubah Style CSS pada Halaman Web.
  4. Dapat Menghapus Element dan Attribut HTML yang sudah ada.
  5. Dapat Mengnambahkan Element dan Attribut HTML.
  6. Dapat memberikan reaksi event pada HTML
  7. Dapat menambahkan reaksi event pada HTML

DOM Methods

Pada Latihan Sebelumnya , function javascript di input dengan nilai paramater seperti berikut ini.

function luasSegitiga (a , t){
    return a*t/2;
}

dengan memasukan nilai alas dengan variable a dan tinggi dengan variable t pada function tersebut akan menampikan nilai jumlah luas segitiga di console, dengan menggunakan DOM Methods dapat menampilkan hasil nilai ke dalam html.

DOM Document

Ada bagian dari methods DOM yang diawali penulisan document.namaMethods, DOM Document dapat melakukan Hal yang dapat dilakukan Document HTML seperti menambahkan dan Mengganti element dan atrribut HTML dan sebagainya, berikut ini adalah contoh dari DOM HTML

<html>
    <head>
        <title>Sample</title>
    </head>
    <body>
        <div id="test">
        </div>
        <script>
            function hello(){
                document.getElementById("test").innerHTML = "<b>Hello World</b>";
            }
            hello()
        </script>
    </body>
</html>

Maka pada atribut id pada element div akan menampilkan tulisan Hello World dalam bentuk bold

pada contoh sebelumnya terdapat DOM document dan DOM element yaitu document.getElementById dan element.innerHTML, yang mana fungsinya document.ElementById sebagai mencari attribut ID dan innerHTML menampilkannya.

dengan menggunakan document DOM kita dapat menjadikan sebagai input dan element DOM sebagai Output untuk contoh program bangun datar berikut ini.

function luasSegitiga (a , t){
    return a*t/2;
    }

Bagaimana dari contoh program bangun datar sebelumnya dapat membuat input dan output dalam bentuk HTML ?

1.Membuat HTML

Buat HTML berikut ini

Buat Struktur HTML dengan 2 input yang hanya dapat memasukan berupa number saja yaitu alas dan tinggi serta juga memberikan attribut id setiap element tag HTML nya.

<html>
    <head>
        <title>Program Bangun Data</title>
    </head>
    <body>
        <script></script>
        <div>
            <input id="alas" type="number" 
            placeholder="Alas"/> 
            <br/>
            <input id="tinggi" type="number" 
            placeholder="Tinggi"/>
            <br/>
            <button onclick=""></button>
        </div>
        <div id="luas"></div>
    </body>
</html>

2. Membuat javascript

Buat penulisan javascript secara internal lalu buat Function Baru mengenai mengenai program dasar bidang datar ganti parameter tapi dengan membuat variable, seperti berikut ini

<script>
    function luasSegitiga(){
        let alas;
        let tinggi;
        let luas = alas * tinggi / 2;
    }
</script>

3. Menambahkan DOM HTML

Tambahkan DOM Document untuk mencari id pada element tag input dan DOM element untuk menampilkan hasil nilai yang telah di jumlahkan

<script>
    function luasSegitiga(){
        let alas = document.getElementById("Alas").value;
        let tinggi = document.getElementById("Tinggi").value;
        let luas = alas * tinggi / 2;
        document.getElementById("luas").innerHTML = luas + " cm<sup>2<sup/>"
        }
</script>
