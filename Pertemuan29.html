HTML DOM (Document Object Model)

HTML DOM adalah teknik manipulasi HTML dimana disaat Halaman Web Dimuat atau tampil , browser membuat Document Object Model. DOM itu adalah struktur HTML berupa document yang dapat dibaca oleh browser. berikut adalah contoh gambaran DOM

DOM HTML YANG DAPAT MELAKUKAN

  1. Dapat Mengganti atau mengubah Element HTML pada Halaman Web.
  2. Dapat Mengganti atau mengubah Attribut HTML pada Halaman Web.
  3. Dapat Mengganti atau mengubah Style CSS pada Halaman Web.
  4. Dapat Menghapus Element dan Attribut HTML yang sudah ada.
  5. Dapat Mengnambahkan Element dan Attribut HTML.
  6. Dapat memberikan reaksi event pada HTML
  7. Dapat menambahkan reaksi event pada HTML

Method dan Property

Method adalah suatu metode yang menyeleksi objek DOM berupa element tag HTML ,atrribut class dan ID, CSS seleksi. Property dalam javascript dapat menggantikan, mengubah, menambahkan dan menghapus suatu DOM pada sebuah halaman. Berikut ini adalah salah satu Contoh Method dan Property 

    <html>
        <head>
            <title>Sample</title>
        </head>
        <body>
            <div id="test">
            </div>
            <script>
                function hello(){
                    document.getElementById("test").innerHTML 
                     = "<b>Hello World</b>";
                }
                hello()
            </script>
        </body>
    </html>

Pada Hasil Kode tersebut menambahkan element HTML pada attribut ID bernama test dengan tulisan Hello World pada browser

pada contoh sebelumnya terdapat method yaitu document.getElementById yang berfungsi untuk mencari element yang memiliki attribut ID dan property element.innerHTML untuk menambahkan element HTML.

dari kesimpulan tersebut yang mana method dapat memberikan nilai seperti input sementara property dapat menampilkan nilai seperti output.

Pada latihan sebelumnya kita telah membuat program sederhana menggunakan function untuk menghitung sebuah bidang datar, pada program tersebut dapat dikembangkan dengan menerapkan method dan property agar program dapat lebih interaktif

function luasSegitiga (a , t){
    return a*t/2;
}

Pada contoh function di atas dapat disimpulkan memiliki dua input berupa alas dan tinggi , lalu nilai dikembalikan dan dijumlahkan agar dapat menampilkan output sebuah nilai dan berikut ini adalah cara bagaimana menerap method dan property pada function tersebut.

3. Menambahkan DOM HTML

Tambahkan method untuk mencari attribut ID pada element tag input dan berikan property untuk menampilkan hasil nilai yang telah di jumlahkan

<script>
    function luasSegitiga(){
        let alas = document.getElementById("alas").value;
        let tinggi = document.getElementById("tinggi").value;
        let luas = alas * tinggi / 2;
        document.getElementById("luas").innerHTML = luas + " cm<sup>2<sup/>"
        }
</script>
