Operator Comparison

Salah satu dari operator javascript untuk membuat suatu perbandingan nilai biasa digunakan untuk sebuah kondisi.

Operator Description
== sama nilai meski type data berbeda
=== harus sama nilai dengan type data yang sama
!= tidak sama nilai meski tipe data berbeda
!== harus tidak sama nilai dengan type data yang sama
> harus lebih besar jika nilai sama maka false
< harus lebih kecil jika nilai sama maka false
>= lebih besar sama dengan jika nilai sama maka true
<= lebih kecil sama dengan jika nilai sama maka true

Operator Logical

adalah operator yang menilai perbandingan sebuah kebenaran dalam keputusan

Operator Dekripsi
&& logical AND
|| logical OR
! logical not

Table Logical

Kondisi A Kondisi B Hasil

false false false

Logical Not

Kondisi Hasil
true false
