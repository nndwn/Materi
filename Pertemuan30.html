Operator Comparison

Salah satu dari operator javascript untuk membuat suatu perbandingan nilai biasa digunakan untuk sebuah kondisi.

Operator Description
== sama nilai meski type data berbeda
=== harus sama nilai dengan type data yang sama
!= tidak sama nilai meski tipe data berbeda
!== harus tidak sama nilai dengan type data yang sama
> harus lebih besar jika nilai sama maka false
< harus lebih kecil jika nilai sama maka false
>= lebih besar sama dengan jika nilai sama maka true
<= lebih kecil sama dengan jika nilai sama maka true

Operator Logical

adalah operator yang menilai perbandingan sebuah kebenaran dalam keputusan

Operator Dekripsi
&& logical AND
|| logical OR
! logical not

Table Logical

Logical AND

Kondisi A Kondisi B Hasil Contoh
True True True

false false false

Logical Or

Kondisi A Kondisi B Hasil
true true true
true false true
false true true
false false false

Logical Not

Kondisi Hasil
true false

3. Menambahkan DOM HTML

Tambahkan method untuk mencari attribut ID pada element tag input dan berikan property untuk menampilkan hasil nilai yang telah di jumlahkan

<script>
    function luasSegitiga(){
        let alas = document.getElementById("alas").value;
        let tinggi = document.getElementById("tinggi").value;
        let luas = alas * tinggi / 2;
        document.getElementById("luas").innerHTML = luas + " cm<sup>2<sup/>"
        }
</script>
