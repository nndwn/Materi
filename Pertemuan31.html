JavaScript if, else, and else if

adalah untuk memberikan suatu kondisi pada suatu pernyataan untuk membuat suatu keputusan. contoh:

if jika benar minum racun ini maka kamu akan mati ,

if else jika hanya sedikit kamu tetap bisa hidup,

else jika tidak kamu tetap akan selalu hidup

Conditional Statements

Seperti penjelasan sebelumnya, didalam bahasa program terdapat 4 Conditional Statement yaitu

  • if akan menjalankan perintah jika kondisi dengan keadaan benar
  • else jika perintah if dalam kondisi false maka perintah else akan menjadi alternatif perintah
  • else if adalah perintah alternatif jika kondisi if adalah false dan else if dalam kondisi benar jika tidak maka akan menjalankan perintah else
  • switch adalah perintah lain dari if jika memiliki banyak kondisi penyataan dalam keputusan

Cara Penulisan Conditional Statement

if (condition) {
    //  eksekusi jika kondisi benar
}

jika ada alternatif lain seperti else

if (condition) {
    //  eksekusi jika kondisi benar
}else {
    //  eksekusi jika kondisi salah
}

Jika ada alternatif kondisi lain seperti else if

if (condition) {
    //  eksekusi jika kondisi benar
}else if (condition){
    //  eksekusi jika kondisi benar apa bila if dalam kondisi salah
}else {
    // eksekusi jika kondisi salah
}
