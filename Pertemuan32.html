JavaScript Object

adalah sebuah type data object

                        const person = {
                            firstName: "John",
                            lastName: "Doe",
                            age: 50,
                            eyeColor: "blue"
                          };

Cara Memanggil data type Object

terdapat 2 cara untuk menampilkan sebuah data type object

  • objectName.propertyName
  • objectName["propertyName"]

objectName.propertyNam

// Buat Type data Object
const person = {
  firstName: "John",
  lastName : "Doe",
  id     :  5566
};

// tampilkan sebuah type data object
document.getElementById("demo").innerHTML =
person.firstName + " " + person.lastName;

objectName["propertyName"]

// Buat Type data Object
const person = {
    firstName: "John",
    lastName : "Doe",
    id     :  5566
    };
                      
// tampilkan sebuah type data object
document.getElementById("demo").innerHTML =
person["firstName"] + " " + person["lastName"];

Membuat Method Menggunakan Object

kita dapat membuat method sendiri dengan menggunakan Object

objectName.methodName()

// Buat Type data object
const person = {
    firstName: "John",
        lastName: "Doe",
        id: 5566,
        fullName: function() {
            return this.firstName + " " + this.lastName;
    }
};
                                    
// Tampilkan method yang telah dibuat
    document.getElementById("demo").innerHTML = person.fullName();

What is this?

javascript this adalah sebuah keyword , pada object type this dapat mendefinisikan bagian suatu object

Aturan Penting Dalam Sebuah Object

jangan gunakan type data strings, Numbers, Booleans menjadi sebuah Object
