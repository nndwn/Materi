JavaScript String

Adalah tipe data text dengan diberikan tanda petik satu ( '' ) atau petik dua (""). 

let carName1 = "Volvo XC60";  // Double quotes
let carName2 = 'Volvo XC60';  // Single quotes

String juga bisa menggambungkan titik dua dan titik satu seperti berikut ini.

let answer1 = "It's alright";
let answer2 = "He is called 'Johnny'";
let answer3 = 'He is called "Johnny"';

Note : String tidak dapat mengabungkan tanda baca titik dua yang sama contoh seperti berikut ini

let text = "We are the so-called "Vikings" from the north.";

untuk mengatasi nya perlu tambahan Backslash di awalnya. seperti berikut ini 

let text = "We are the so-called \"Vikings\" from the north.";

berikut adalah table tanda baca backslash yang dapat digunakan

Code Result Description
\' ' Single Quote
\" " Double Quote
\\ \ Backslash

Selain itu tipe Data String dapat menjumlah seluruh isi text dan dijadikan sebuah tipe data number dengan menggunakan property

let text = "saya tampan sekali";
console.log(text.length); // menampilkan 18 huruf dan spasi

note: tanda spasi di string di hitung juga nya

String Method dan Property

banyak method dan property di javascript untuk memanipulasi data pada variable string.

variable string dapat di ambil setiap katanya dengan menggunakan method di bawah ini

  • slice(start, end)
  • substring(start, end)
  • substr(start, length)

JavaScript String slice()

berikut ini adalah cara kerja method slice()

let str = "Apple, Banana, Kiwi";
console.log(str.slice(7, 13));
//hasil Banana

let str = "Apple, Banana, Kiwi";
console.log(str.slice(-12, -6));
//hasil Banana dengan mengambil dari kiri ke kanan
    
let str = "Apple, Banana, Kiwi";
console.log(str.slice(7));
//Hasil Banana, Kiwi            
        
let str = "Apple, Banana, Kiwi";
console.log (str.slice(-12));
//Hasil Banana, Kiwi

Note : pada method slice() dapat mengambil teks string positif dari kanan , jika nilai nya minus maka mengambil teks nya dari kiri

JavaScript String substring()

sama seperti method slice() fungsi nya bedanya subtring tidak dapat mengambil dengan nilai minus, berikut ini adalah cara nya

    let contohSubtring = "Apple, Banana, Kiwi";
    console.log(contohSubtring.substring(7, 13));

note : method substring() tidak dapat menampikan nilai di bawah 0

JavaScript String substr()

sama fungsinya dengan method sebelumnya tapi perbedaannya di parameter akhirnya adalah jumlah text yang mau di tampilkan 

let contohSubstr = "Apple, Banana, Kiwi";
console.log(substr(7,6));

let contohSubstr = "Apple, Banana, Kiwi";
console.log(substr(7));

Note : Method substr() jika di buat dengan nilai negatif maka teks string yang akan tampil dari kiri.

Mengganti isi Content String

string dapat diganti teks di dalam variablenya dengan menggunakan method replace() . seperti berikut ini

    let contohReplace = "Apple, Apple, Banana, Kiwi";
    console.log(contohReplace.replace("Apple", "Watermelon"));

Note : pada method replace hanya bisa menganti teks yang di awal saja selain itu method ini sifat nya sensi case memperhatikan besar kecil huruf

Menganti format teks string

di javascript terdapat 2 method untuk mengubah format text

  • toUpperCase() // membuat huruf besar semua
  • toLowerCase() // membuat huruf kecil semua
    let namaHewan1 = "kelinci";
    let namaHewan2 = "HARIMAU";
    console.log(namaHewan1.toUpperCase());
    console.log(namaHewan2.toLowerCase());

note: perlu diingat semua method mengubah text di jelaskan sebelumnya tidak mengubah string yang original

Mangabung dua atau tiga string yang berbeda

dengan menggunakan method concat() dapat mengabungkan dua atau tiga variable yang berbeda menjadi satu, berikut contohnya

    let text1 = "Hello";
    let text2 = "World";
    let text3 = text1.concat(" ", text2); 
    console.log(text3); //akan tampilan Hello World

pada contoh tersebut sama hal nya dengan

    let text3 = text1 + " " + text2;

Menhitung setiap kalimat atau kata pada string

dengan menggunakan kan method split() dapat memisahkan setiap kata pada sebuah kalimat pada string

    let contohKoma = "buaya,kucing,harimau";
    console.log(contohKoma.split(",").length);
    let contohSpace = "susah sekali belajar javascript";
    console.log(contohSpace.split(" ").length);

Mengambil salah satu huruf pada string

pada javascrip terdapat dua cara mengambil salah satu huruf pada string, menggunakan method dan property, seperti berikut ini.

        let contohChar ="Hello World";
        console.log(contohChar.charAt(0)); //menampilkan huruf dimulai dari nol yaitu H , jika di ubah jadi 2 maka akan tampil huruf

cara kedua menggunakan access property

        let contoChar2 ="Hello World";
        console.log(contoChar2[0]);

note : meskipun menggunakan bracket seperti array tapi itu bukan array
