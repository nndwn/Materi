Array

Adalah spesial variable yang dapat menampung banyak nilai.

Mengapa gunakan Array

Jika seandainya banyak sebuah variable yang perlu di tampung sebuah nilai, contoh berikut ini. 

    let car1 = 'toyota';
    let car2 = 'honda';
    let car3 = 'bmw';

dengan array kamu tidak perlu membuat banyak sebuah variable dengan cara satu persatu.

Cara Penulisan Array

    const array_name = [item1, item2, item3 ];

Contoh Array

Berikut ini adalah cara penerapannyan

    const cars = ["toyota", "honda", "BMW"];

    const cars = [
        "toyota",
        "honda",
        "BMW"
    ]

Memanggil Array

berikut adalah cara memanggil salah satu array 

    const cars = ["toyota", "honda", "BMW"];
    console.log (cars[0]);

dan berikut ini adalah cara bagaimana menampilkan seluruh nilai array

    const cars = ["toyota", "honda", "BMW"];
    console.log (cars);

Mengganti Nilai Array

berikut ini adalah cara bagaimana mengganti sebuah nilai yang telah dibuat.

    const cars = ["toyota", "honda", "BMW"];
    cars[0] = "suzuki";
    console.log(cars);

Array adalah type data Object

Array adalah spesial type object, di javascript array di denefinisikan salah satu type data object, kita dapat memeriksa sendiri dengan perintah berikut ini.

    const cars = ["toyota", "honda", "BMW"];
    console.log(typeof(cars));

Array Element Bisa di letekkan apa saja

Array dapat diletkan apapun seperti function , array di dalam array, dan type data lainnya.

Perbedaan Array dan Objects

Objects

    const cars = {car1 : "toyota", car2: "honda",car3:"BMW" };
    console.log(cars.car1);

Array

    const cars = ["toyota", "honda", "BMW"]
    console.log(cars[0]);

Perbedaan Array dan Objects

dari kode sebelumnya dapat disimpulkan memiliki sebuah perbedaan yaitu

  • Array menggunakan Penomoran untuk menampilkan indexnya
  • Object menggunakan Penamaan untuk menampilkan indexnya

Kapan Gunakan Array dan Objects

  • jika data banyak element teks (string) gunakan object
  • Jika data banyak element Nomor(Number) gunakan array

1. Array Methods

1.1 Convert Array Menjadi String

Menampilkan isi di dalam index array harus menjadi array

  • toString()
  • join()

1.2 Menghapus menambahkan array

Array dapat menambahkan dan menghampus element di dalam index array

  • pop();//menghapus element terakhir pada index array
  • push();//menambahkan element terakhir pada index array
  • shift(); //menghapus element awal pada index array
  • unshift(); //menambahkan element awal pada index array

1.3 Mengabungkan dua Array berbeda

  • concat();
const myGirls = ["Cecilie", "Lone"];
const myBoys = ["Emil", "Tobias", "Linus"];

const myChildren = myGirls.concat(myBoys);

1.4 Menyisipkan dan menghapus array

  • splice()

fungsi splice() dapat menambahkan dan menghapus array

1.5 Memotong array

  • slice()

fungsi slice dapat memotong array

1.6 Mengurut Array

  • sort(); mengurutkan berdasarkan (Ascending)
  • reverse(); mengurutkan bersarkan (Descending)
