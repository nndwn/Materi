Kelvin to Fahrenheit: F = 9/5(K - 273) + 32 or F = 1.8(K - 273) + 32

Celsius to Fahrenheit: F = 9/5(C) + 32 or F = 1.80(C) + 32

Celsius to Kelvin: K = C + 273 (or K = C + 271.15 to be more precise)

Fahrenheit to Celsius: C = (F - 32)/1.80
